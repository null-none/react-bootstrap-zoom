# react-bootstrap-zoom
Images zoom on hover as react bootstrap components


## Install

```bash
npm install react-bootstrap-zoom
```

## Example

```js
import ZoomImage from 'react-bootstrap-zoom';

<ZoomImage src='/path/images.png' />

```
